# Bible Tracker

A web application for tracking personal Bible reading progress. Users can log daily reading sessions, visualize progress across books and testaments, and optionally follow structured reading plans.

## Features (Planned)

- **Reading Log** — Record daily reading by book, chapter, and verse with support for multiple passages per session
- **Progress Dashboard** — Visual gauges for Old Testament, New Testament, and whole-Bible completion, plus a GitHub-style reading activity graph
- **Reading Plans** — Optional curated plans with daily readings and one-click completion tracking
- **Multi-Canon Support** — Protestant (66 books), Catholic (73 books), and Jewish/Tanakh canons, with optional Apocrypha for Protestant users

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python / FastAPI |
| Database | PostgreSQL |
| ORM | SQLAlchemy 2.0 (async) + Alembic |
| Auth | JWT |
| Frontend | React + Vite + Tailwind CSS |
